# DSCI551-final-project
Softwares needed: python interface, MySQL workbench with established local connection
requirements.txt is given to install python dependencies
Add MySQL localhost username and password to DB_CONFIG dictionary and upload the CSV files on MySQL
Use a gemini 2-0 flash API key and add it to "config.json" file
Run the py file "app.py" in terminal using: "streamlit run app.py"
The three databases are stored in data folder
In the streamlit app, select the database that you are willing to use and write a natural language query
